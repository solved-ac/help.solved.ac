<ExternalLink href="https://solved.ac/class">CLASS</ExternalLink>

**CLASS**는 solved.ac가 큐레이션한 문제들을 풀어서 취득할 수 있는 등급입니다. CLASS를 취득하면 최대 250까지의 보너스 레이팅을 받을 수 있습니다.

- 참고: 가이드 [「티어와 AC 레이팅」](/ko/stats/ac-rating)

## 단계

CLASS에는 총 10단계가 있습니다. 각 단계마다는 solved.ac가 미리 정해 둔 36&ndash;48개의 문제들이 있습니다. 이 문제들 중 아무 문제나 골라, 각 CLASS의 **취득 조건**에 해당하는 문제 수만큼을 해결하면 그 CLASS를 취득할 수 있습니다.

- CLASS 9와 10은 이전 단계의 CLASS를 취득해야 취득할 수 있습니다.

- **CLASS 1**: 프로그래밍 문제해결 초심자가 문제해결에 익숙해질 수 있도록 하는 단계
- **CLASS 2**: 기초적인 자료 구조, 알고리즘 및 컴퓨팅 사고력을 이용해 문제를 해결해야 하는 단계
- **CLASS 3&ndash;4**: 코딩 테스트 및 현업에서 사용되는 컴퓨터과학 학부 수준의 자료 구조, 알고리즘 및 컴퓨팅 사고력을 이용해 문제를 해결해야 하는 단계
- **CLASS 5&ndash;6**: 프로그래밍 대회에서 등장하는 여러 기반 지식을 이용해 문제를 해결해야 하는 단계
- **CLASS 7&ndash;10**: 프로그래밍 대회에서 등장하는 심화 지식을 이용해 문제를 해결해야 하는 단계

## 장식

특정 조건을 만족하면 취득한 CLASS에 **장식**이 추가됩니다.

|                                | 장식 |                 조건                 |
| :----------------------------: | :--: | :----------------------------------: |
|      <ClassIcon c={1} />       | 없음 |             CLASS를 취득             |
| <ClassIcon c={1} d="silver" /> | 은장 | CLASS 내의 에센셜 문제들을 전부 해결 |
|  <ClassIcon c={1} d="gold" />  | 금장 |    CLASS 내의 문제들을 전부 해결     |

장식을 취득하면 랭킹 페이지 등 일부 페이지에서 CLASS가 $x$ 대신 $x$⁺(은장), $x$⁺⁺(금장)으로 표시됩니다. 레이팅 보너스는 없습니다.
